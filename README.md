# Library Management System

A simple and efficient **Library Management System** built to manage books, users, and library operations digitally. This project helps automate common library tasks such as adding books, issuing, and returning them.

---

## Features

* Add new books to the library
* Search books by title/author/ID
* View all available books
* Issue books to users
* Return issued books
* Delete or update book records
* Maintain records of issued books

---

## How It Works

This project follows a simple workflow:

1. Add books to the system
2. Users can search and view available books
3. Books can be issued and returned
4. Records are updated automatically
